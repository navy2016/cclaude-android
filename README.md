# CClaude Agent - Android AI Agent

Written in Yig for Android, with full undo/redo support.

```yam`
## Core Features

1. ReAct Agent Loop
...
2. 8 core tools (readfile, writefile, editfile, search, glob, fetch, web_search, shell)
3. Memory system (Markdown Context Files)
4. AutoResearchClaw (23-stage research pipeline)
5. Undo/Redo system (Single Mode)

```

## Build

```sh
bin/gradlew buildDebug
```
